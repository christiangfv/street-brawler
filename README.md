## 🕹️ CONTROLS

### 🔵 Player 1 (Keyboard)

| Action | Key |
|--------|-----|
| Move Left | `A` |
| Move Right | `D` |
| Jump | `W` |
| Crouch/Block | `S` |
| Light Punch | `U` |
| Medium Punch | `I` |
| Heavy Punch | `O` |
| Light Kick | `J` |
| Medium Kick | `K` |
| Heavy Kick | `L` |

### 🔴 Player 2 (Arrows + Numpad)

| Action | Key |
|--------|-----|
| Move Left | `←` |
| Move Right | `→` |
| Jump | `↑` |
| Crouch/Block | `↓` |
| Light Punch | `Numpad 4` |
| Medium Punch | `Numpad 5` |
| Heavy Punch | `Numpad 6` |
| Light Kick | `Numpad 1` |
| Medium Kick | `Numpad 2` |
| Heavy Kick | `Numpad 3` |

### 📱 Mobile / Touch

Full touch controls with on-screen **D-Pad** and **6-button layout**!

## ⚡ GAME MECHANICS

| Mechanic | Description |
|----------|-------------|
| **HP System** | Each fighter has 100 HP. Reduce your opponent to 0! |
| **Best of 3** | Win 2 rounds to claim victory 🏆 |
| **6-Button Combat** | Light/Medium/Heavy Punches & Kicks — mix it up! |
| **Blocking** | Hold crouch when getting hit to reduce damage |
| **Special Moves** | Heavy attacks have extra range and damage |
| **Hit Stun** | Landing hits staggers your opponent briefly |
| **90-Second Rounds** | If time runs out, highest HP wins the round |

## 🚀 HOW TO PLAY

### Option 1: Play Online
👉 **[Click here to play](https://christiangfv.github.io/street-brawler/)**

### Option 2: Run Locally

```bash
# Clone the repo
git clone https://github.com/christiangfv/street-brawler.git
cd street-brawler

# Serve it (pick one)
npx serve .
# or
python3 -m http.server 8080

# Open http://localhost:8080
```

That's it. No `npm install`. No webpack. No bullshit.
