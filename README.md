## 🕹️ CONTROLS

<div align="center">

<img src="assets/spr_chris_special.png" height="120" align="right">

</div>

### 🔵 Player 1

| Action | Key |
|--------|-----|
| Move | `A` / `D` |
| Jump | `W` |
| Crouch / Block | `S` |
| Punches | `U` `I` `O` |
| Kicks | `J` `K` `L` |

### 🔴 Player 2

| Action | Key |
|--------|-----|
| Move | `←` `→` |
| Jump | `↑` |
| Crouch / Block | `↓` |
| Punches | `Num 4` `5` `6` |
| Kicks | `Num 1` `2` `3` |

### 📱 Mobile

Full touch controls — D-Pad + 6-button layout. Works in portrait and landscape.

## 🎨 FEATURES

- 🎭 **24 Unique Fighters** with custom pixel art sprites
- 🗺️ **Multiple Stages** — Zapallar, Colchagua, Valle Bravo and more
- 🎵 **Chiptune BGM** — Dynamic 8-bit music per scene
- 📱 **Fully Responsive** — Desktop, tablet, mobile
- 🎮 **1P vs AI or 2P Local** — Solo or couch co-op
- ⚡ **Zero Dependencies** — Pure vanilla JS, no build step

### Online

```bash

cd street-brawler
npx serve .
```
